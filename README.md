# 包材庫存管理 PWA

這是可安裝到 iPhone 主畫面的包材庫存管理 PWA，資料使用瀏覽器 `localStorage` 儲存，離線時仍可開啟與修改庫存。

## 安裝依賴

```bash
npm install
```

## 啟動

```bash
npm start
```

啟動後打開：

```text
http://localhost:3000
```

## iPhone 加入主畫面

1. 用 Safari 打開網站網址。
2. 點 Safari 的分享按鈕。
3. 選擇「加入主畫面」。
4. 從主畫面打開「包材庫存」App。

## 開啟庫存通知

1. 從加入主畫面後的 App 打開網站。
2. 點「開啟庫存通知」。
3. 允許通知後，畫面會顯示「已開啟庫存通知」。
4. 新增或儲存包材時，若現有庫存小於或等於最低庫存，會發送本機通知。

通知內容格式：

```text
包材庫存不足
XXX 庫存不足，目前庫存 XX，最低庫存 XX，請安排補貨。
```

## iOS PWA 通知限制

iOS 的 PWA 通知必須符合以下條件：

- 使用 Safari 將網站「加入主畫面」。
- 從主畫面打開 App。
- 使用者點擊「開啟庫存通知」並允許通知。
- 如果裝置或瀏覽器不支援 Notification API，畫面會顯示「此裝置不支援本機通知，請改用畫面內提醒。」

## 避免重複通知

系統會使用 `localStorage` 記錄每個包材已通知的庫存數量。

- 同一包材同一庫存數量已通知過，不會重複通知。
- 庫存恢復正常後，會清除該包材通知紀錄。
- 庫存再次低於最低庫存，或庫存數字變動後，會再次通知。
